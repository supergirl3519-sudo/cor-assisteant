## 🎯 Что это?

ЦОР-Ассистент — это бесплатное веб-приложение, которое помогает учителям создавать качественные цифровые образовательные ресурсы за считанные секунды с помощью искусственного интеллекта.

### Поддерживаемые типы ЦОР:
- 📋 **Конспекты уроков** — с целями, этапами, методами, оцениванием
- 📝 **Тесты и контрольные** — с ответами и пояснениями (формат ЕГЭ/ОГЭ)
- 📊 **Презентации** — описание слайдов с содержанием
- ✏️ **Самостоятельные работы** — для работы на уроке
- 🃏 **Карточки-задания** — для индивидуальной и групповой работы
- 🔠 **Кроссворды** — с вопросами, ответами и сеткой
- 📄 **Рабочие листы** — для печати и практики
- 📅 **Календарно-тематические планы** — планирование на четверть/год
- 📑 **Рабочие программы** — с пояснительной запиской
- ✍️ **Диктанты и проверочные** — с грамматическим заданием
- 📖 **Темы для эссе** — с планом и критериями
- 🔬 **Проектные задания** — с этапами исследования

## 🚀 Быстрый старт

### 1. Развертывание на GitHub Pages (бесплатно)

```bash
# 1. Создайте новый репозиторий на GitHub
# 2. Загрузите все файлы из этого проекта
# 3. Перейдите в Settings → Pages
# 4. Выберите ветку main и папку / (root)
# 5. Сайт будет доступен по адресу: https://yourusername.github.io/cor-assistant
```

### 2. Получение API ключа (бесплатно)

1. Перейдите на [openrouter.ai](https://openrouter.ai)
2. Зарегистрируйтесь (можно через Google)
3. Перейдите в раздел **Keys** → **Create Key**
4. Скопируйте ключ (начинается с `sk-or-v1-...`)
5. Вставьте ключ в настройки приложения

> 💡 **Бесплатные модели** доступны без привязки карты. Лимит: 20 запросов/мин, 50-200 запросов/день.

### 3. Использование

1. Откройте сайт
2. Введите API ключ в разделе **Настройки API**
3. Выберите предмет, класс, тип ЦОР и тему
4. Нажмите **✨ Сгенерировать**
5. Скачайте результат в формате Markdown или скопируйте

## 🛠 Технологии

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **API:** OpenRouter (единый доступ к 400+ моделям ИИ)
- **Хранение:** LocalStorage (всё хранится локально в браузере)
- **Хостинг:** GitHub Pages (бесплатно)

## 🔒 Безопасность и приватность

- 🔑 API ключ хранится **только в вашем браузере** (LocalStorage)
- 🚫 Ключ **не отправляется** на сторонние серверы (кроме OpenRouter)
- 📝 История материалов хранится **локально** на вашем устройстве
- 🌐 Весь код открытый — вы можете проверить его самостоятельно
