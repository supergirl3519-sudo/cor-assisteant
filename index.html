🔑 Настройки API

Для генерации ЦОР используется ИИ через API. Вы можете использовать бесплатный ключ OpenRouter или свой.

Получить бесплатный ключ на OpenRouter →

📖 Как получить бесплатный API ключ:

  1. Перейдите на openrouter.ai/keys
  2. Зарегистрируйтесь (можно через Google/GitHub)
  3. Нажмите «Create Key»
  4. Скопируйте ключ и вставьте сюда

⚠️ Ключ хранится только в вашем браузере и не отправляется на наши серверы.
